# Changelog

- Replaced direct recommendations with provisional Current Judgements.
- Added six independent Advisors and an Agora deliberation step.
- Added Understanding Before Guidance to the daily flow.

- Simplified onboarding to name-only relationship-first onboarding.
- Migrates legacy v0.4.2 local data into the v0.5 store.
- Fixed the duration fallback bug in the former decision engine.
